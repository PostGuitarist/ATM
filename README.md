# ATM
Python script that acts as an ATM, integrates Turtle Graphics. 

<h1>Prerequisites</h1> 
<ul>
  <li>Python 3.x</li>
  <li>Python Turtle</li>
 </ul>

<h1>Installation</h1>
<ol>
  <li>Extract the file to your desktop</li>
  <li>Navigate to the file using the terminal</li>
  <li>Use Python 3.x to run the file</li>
 </ol>
